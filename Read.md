# PRODIGY_WD_03 — Tic-Tac-Toe Web Application

## 🎮 Project Overview
This project was completed as **Project 3** of the **Web Development Internship at Prodigy InfoTech**.

The objective of this task was to build an **interactive Tic-Tac-Toe web application** using HTML, CSS, and JavaScript by handling user interactions, tracking game state, and checking winning conditions.

---

## 🎯 Task Objective
- Build a browser-based Tic-Tac-Toe game
- Handle user clicks and turns
- Track game state dynamically
- Detect win and draw conditions
- Improve JavaScript logic and DOM manipulation skills

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the game board  
- **CSS3** – Styling and layout  
- **JavaScript** – Game logic and interactivity  

---

## ✨ Features
- Two-player gameplay (X and O)
- Turn-based logic
- Automatic win detection
- Draw condition handling
- Reset game option
- Clean and responsive UI
